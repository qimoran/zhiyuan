# 考研择校推荐系统 - Figma 设计完整提示词

## 📋 项目基本信息

**项目名称**：考研择校推荐系统  
**项目简介**：为考研学生提供科学、数据驱动的高校选择建议  
**用户群体**：大三、大四考研学生（18-25岁）  
**应用场景**：Web 应用（支持 PC、平板、手机响应式）  
**核心功能**：学校查询、专业浏览、成绩评估、智能推荐、图表展示  

---

## 🎨 设计系统总体策略

### 设计理念
- **专业严谨**：体现考研的严肃性和重要性
- **温暖易用**：降低学生的焦虑感，提供友好的交互体验
- **数据可视化**：清晰展示复杂的分数数据和推荐逻辑
- **渐进式引导**：用户逐步了解系统功能，建立信任感

### 目标用户心理
- 考试焦虑：需要可靠的数据支撑
- 决策困难：需要清晰的推荐和对比
- 时间紧张：需要快速获取关键信息
- 缺乏经验：需要充分的帮助和解释

### 阴影系统（Shadow）

**阴影组合**：采用分层阴影，增加深度感

**无阴影（Flat）**
- 用于主体卡片、输入框

**轻阴影（Subtle）**
```
Offset: 0px 1px 2px
Blur: 4px
Color: rgba(0,0,0,0.05)
Spread: 0px
```
用途：普通卡片、列表项悬停态

**中阴影（Default）**
```
Offset: 0px 4px 6px
Blur: 12px
Color: rgba(0,0,0,0.08)
Spread: 0px
```
用途：浮动卡片、模态框背景

**深阴影（Elevated）**
```
Offset: 0px 10px 20px
Blur: 25px
Color: rgba(0,0,0,0.1)
Spread: 0px
```
用途：模态框、下拉菜单、模态对话框

**超深阴影（High Elevation）**
```
Offset: 0px 20px 40px
Blur: 40px
Color: rgba(0,0,0,0.12)
Spread: 0px
```
用途：顶部导航、固定浮层

## 🎬 动画与交互规范

### 页面过渡动画
- 淡入/淡出：300ms，缓动函数 `ease-in-out`
- 滑动进入：从右侧滑入 400ms
- 后退返回：从左侧滑出 300ms

### 组件交互
- 按钮点击：按下态 0.9 倍缩放，100ms 回弹
- 卡片悬停：阴影加强，上移 4px，300ms 过渡
- 加载状态：旋转加载图标，无限循环

### 表单交互
- 聚焦输入框：边框高亮青色，背景浅青，200ms 过渡
- 输入框出错：边框变红 `#ef4444`，配合摇晃动画 300ms
- 表单验证成功：绿色对勾图标，缓进 200ms

### 分页加载
- 向下滚动到底部时：自动加载下一页
- 加载中：加载指示器在底部居中
- 加载完成：淡入新卡片，200ms 动画
