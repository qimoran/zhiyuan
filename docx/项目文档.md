# 项目名称

重庆高校考研择校推荐系统

# 项目背景

我们是大学本科大数据专业大三的学生，小组共四人。我们现在要做一个为期三个星期的实践项目，项目方向从原来的“高考志愿填报辅助”调整为“考研择校推荐辅助”。项目仍然面向重庆高校范围，重点围绕重庆地区高校的硕士研究生招生信息、复试分数线、拟录取名单、招生专业目录等公开资料，构建从数据收集、数据清洗、数据存储、大数据分析、可视化展示到择校推荐和 AI 建议报告生成的完整数据应用链路。

本项目的数据来源以各高校研究生院官网、学院官网、研招网公开信息为主。考虑到很多考研数据以 PDF、Excel、网页公告等形式发布，项目第一阶段不追求完全自动爬取所有数据，而是采用“人工收集官方文件 + 建立来源索引 + 半自动解析 + 人工校对 + 标准化入库”的方式，保证数据来源可靠、可追溯，也更符合课程实训的实际工作量。

系统面向准备报考重庆高校研究生的学生。用户输入自己的初试总分、单科成绩、目标专业门类、意向学校层次等信息后，系统结合历年复试线、拟录取分数、招生人数、专业热度等指标，给出“冲刺、稳妥、保底”三类院校或专业推荐，并生成一份考研择校建议报告。

# 项目功能

1. 官方资料收集：手动从重庆高校研究生院官网、学院官网、研招网等公开渠道收集硕士研究生招生专业目录、复试分数线、拟录取名单、招生简章、调剂公告等 PDF、Excel 或网页资料，并记录来源链接、发布日期、采集人员和处理状态。
2. 数据来源管理：建立 `pdf_source_index` 或类似的数据来源登记表，记录学校名称、年份、文档类型、文件路径、来源 URL、发布时间、采集时间、提取状态和备注，保证后续数据能够追溯到官方原始文件。
3. PDF 与表格数据处理：使用 Python 工具对文字版 PDF、Excel 和网页表格进行半自动提取；对于扫描版 PDF 或格式复杂的文件，先进行人工校对，再整理为统一的结构化数据。
4. 数据清洗与标准化：对学校名称、学院名称、专业代码、专业名称、研究方向、学习方式、考试科目、招生人数、复试线、拟录取分数等字段进行清洗和标准化，处理重复数据、缺失值和不同学校之间的格式差异。
5. 数据存储：将清洗后的考研招生数据、复试线数据、拟录取数据、专业目录数据写入 MySQL 数据库，并同步到 Hadoop 平台的 HDFS/Hive 中，支撑后续分析和查询。
6. 数据分析：使用 Spark、Pandas、NumPy 对存储数据进行分析，统计重庆高校不同专业近几年复试线变化、拟录取最低分和平均分变化、招生人数变化、专业热度、竞争程度等指标。
7. 可视化展示：使用 ECharts 展示分析结果，包括学校分布、专业热度排行、复试线趋势图、拟录取分数趋势图、招生人数变化图、不同学校同类专业对比图等。
8. 考研择校推荐：用户输入初试总分、政治、英语、业务课等单科成绩，以及目标专业门类、学习方式、意向学校层次后，系统根据国家线、院校复试线、拟录取分数、招生人数和专业竞争度，生成“冲刺、稳妥、保底”推荐列表。
9. 推荐理由解释：每条推荐结果需要展示推荐依据，例如总分高出复试线多少分、单科是否安全、近几年拟录取最低分和平均分情况、招生人数是否稳定、专业竞争是否较高等。
10. AI 择校建议报告：系统将用户输入信息和推荐结果传给大模型，生成一份考研择校建议报告，内容包括考生基本情况分析、推荐梯度说明、风险提示、备考建议和调剂关注方向。报告必须基于系统提供的真实数据生成，不得承诺一定录取。

# 数据范围

本项目第一版建议控制数据范围，先保证系统可运行、可演示、可答辩。

1. 地区范围：重庆高校。
2. 学校范围：优先选择 8 到 10 所重庆高校，例如重庆大学、西南大学、重庆邮电大学、重庆交通大学、重庆医科大学、西南政法大学、重庆师范大学、重庆工商大学、四川外国语大学、重庆理工大学等。
3. 年份范围：优先收集近 3 年数据。
4. 专业范围：优先选择数据较容易获取、报考热度较高的专业方向，例如计算机、电子信息、教育、法学、医学、管理、机械、土木等。
5. 文档类型：招生专业目录、复试分数线、拟录取名单为核心数据；招生简章、调剂公告、报录比作为补充数据。

# 推荐规则初步思路

考研推荐不再使用高考的一分一段表和位次差，而是采用多指标综合判断。

1. 分数线判断：判断用户总分和单科成绩是否超过国家线、院校线或专业复试线。
2. 总分差计算：计算用户总分与历年复试线、拟录取最低分、拟录取平均分之间的差值。
3. 单科安全判断：判断政治、英语、业务课等单科成绩是否存在压线风险。
4. 招生人数分析：分析目标专业近几年招生人数是否稳定，是否存在明显扩招或缩招。
5. 专业热度分析：根据拟录取人数、公开报录比、调剂情况等指标估计专业竞争程度。
6. 推荐分档：根据综合评分将目标院校或专业划分为“冲刺、稳妥、保底”三类。
7. 风险提示：对数据缺失、招生人数过少、复试线波动大、拟录取分数偏高的专业进行明确提示。
