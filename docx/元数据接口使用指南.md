# 元数据接口使用指南

## 概述

为了方便前端构建下拉列表，系统提供了 4 个元数据枚举接口。这些接口返回系统中可用的专业门类、学位类型、学习方式、学校层次等数据。

---

## 1. 专业门类列表

### 接口地址
```
GET /api/metadata/major-categories
```

### 功能说明
- 返回系统中所有可用的专业门类
- 数据来源：`majors` 表、`score_lines` 表（专业线和国家线）
- 合并去重后提供完整列表

### 响应示例
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "combined": [
      "哲学",
      "经济学",
      "法学",
      "教育学",
      "文学",
      "理学",
      "工学",
      "医学",
      "管理学",
      "艺术学",
      "计算机技术",
      "电子信息",
      "机械",
      "会计",
      "工商管理",
      ...
    ],
    "from_majors": [...],          // 从 majors 表提取
    "from_score_lines": [...],     // 从专业线提取
    "from_national_lines": [...]   // 从国家线提取
  }
}
```

### 前端使用示例（HTML）
```html
<select name="major_category" id="major_category">
  <option value="">-- 请选择专业门类 --</option>
  <!-- 通过 JavaScript 动态填充 -->
</select>

<script>
fetch('/api/metadata/major-categories')
  .then(res => res.json())
  .then(data => {
    const select = document.getElementById('major_category');
    data.data.combined.forEach(category => {
      const option = document.createElement('option');
      option.value = category;
      option.textContent = category;
      select.appendChild(option);
    });
  });
</script>
```

### 当前数据
- **总数**：573 个专业门类
- **包含**：学科门类（哲学、经济学、法学等）+ 专业学位类别（电子信息、机械等）+ 具体专业（计算机技术、会计等）

## 2. 学位类型枚举

### 接口地址
```
GET /api/metadata/degree-types
```

### 功能说明
返回学位类型枚举值。

### 响应示例
```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "value": "academic",
      "label": "学术学位（学硕）"
    },
    {
      "value": "professional",
      "label": "专业学位（专硕）"
    }
  ]
}
```

### 前端使用示例（HTML）
```html
<select name="degree_type" id="degree_type">
  <option value="">-- 请选择学位类型 --</option>
</select>

<script>
fetch('/api/metadata/degree-types')
  .then(res => res.json())
  .then(data => {
    const select = document.getElementById('degree_type');
    data.data.forEach(item => {
      const option = document.createElement('option');
      option.value = item.value;
      option.textContent = item.label;
      select.appendChild(option);
    });
  });
</script>
```

## 前端集成示例

### 完整的推荐表单示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>考研择校推荐</title>
</head>
<body>
  <h2>考研择校推荐系统</h2>
  
  <form id="recommend-form">
    <div>
      <label>目标年份：</label>
      <select name="target_year" required>
        <option value="2024">2024</option>
        <option value="2025">2025</option>
        <option value="2026" selected>2026</option>
      </select>
    </div>

    <div>
      <label>专业门类：</label>
      <select name="major_category" id="major_category" required>
        <option value="">-- 请选择 --</option>
      </select>
    </div>

    <div>
      <label>学位类型：</label>
      <select name="degree_type" id="degree_type">
        <option value="">-- 请选择 --</option>
      </select>
    </div>

    <div>
      <label>学习方式：</label>
      <select name="study_mode" id="study_mode">
        <option value="">-- 请选择 --</option>
      </select>
    </div>

    <div>
      <label>学校层次：</label>
      <select name="school_level" id="school_level" multiple>
        <!-- 多选 -->
      </select>
    </div>

    <div>
      <label>总分：</label>
      <input type="number" name="total_score" min="0" max="500" required>
    </div>

    <div>
      <label>政治：</label>
      <input type="number" name="politics_score" min="0" max="150" required>
    </div>

    <div>
      <label>英语：</label>
      <input type="number" name="english_score" min="0" max="150" required>
    </div>

    <div>
      <label>专业课一：</label>
      <input type="number" name="subject_one_score" min="0" max="150" required>
    </div>

    <div>
      <label>专业课二：</label>
      <input type="number" name="subject_two_score" min="0" max="150" required>
    </div>

    <button type="submit">获取推荐</button>
  </form>

  <div id="result"></div>

  <script>
    // 加载专业门类
    fetch('/api/metadata/major-categories')
      .then(res => res.json())
      .then(data => {
        const select = document.getElementById('major_category');
        data.data.combined.forEach(category => {
          const option = document.createElement('option');
          option.value = category;
          option.textContent = category;
          select.appendChild(option);
        });
      });

    // 加载学位类型
    fetch('/api/metadata/degree-types')
      .then(res => res.json())
      .then(data => {
        const select = document.getElementById('degree_type');
        data.data.forEach(item => {
          const option = document.createElement('option');
          option.value = item.value;
          option.textContent = item.label;
          select.appendChild(option);
        });
      });

    // 加载学习方式
    fetch('/api/metadata/study-modes')
      .then(res => res.json())
      .then(data => {
        const select = document.getElementById('study_mode');
        data.data.forEach(item => {
          const option = document.createElement('option');
          option.value = item.value;
          option.textContent = item.label;
          select.appendChild(option);
        });
      });

    // 加载学校层次
    fetch('/api/metadata/school-levels')
      .then(res => res.json())
      .then(data => {
        const select = document.getElementById('school_level');
        data.data.forEach(item => {
          const option = document.createElement('option');
          option.value = item.value;
          option.textContent = item.label;
          select.appendChild(option);
        });
      });

    // 表单提交
    document.getElementById('recommend-form').addEventListener('submit', function(e) {
      e.preventDefault();
      
      const formData = new FormData(e.target);
      const data = Object.fromEntries(formData);
      
      // 转换数字类型
      ['target_year', 'total_score', 'politics_score', 'english_score', 
       'subject_one_score', 'subject_two_score'].forEach(field => {
        if (data[field]) {
          data[field] = parseInt(data[field]);
        }
      });

      // 调用推荐接口
      fetch('/api/recommend', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify(data)
      })
      .then(res => res.json())
      .then(result => {
        document.getElementById('result').innerHTML = 
          '<pre>' + JSON.stringify(result, null, 2) + '</pre>';
      });
    });
  </script>
</body>
</html>
```

## 数据说明

### 专业门类（573 个）

包含三类数据：
1. **学科门类**（13个）：哲学、经济学、法学、教育学、文学、历史学、理学、工学、农学、医学、军事学、管理学、艺术学
2. **专业学位类别**（约40个）：电子信息、机械、材料与化工、资源与环境、能源动力、土木水利、生物与医药、交通运输、金融、应用统计、法律、教育、体育、翻译、新闻与传播、工程管理、工商管理、会计、审计等
3. **具体专业**（约520个）：计算机技术、计算机科学与技术、软件工程、信息与通信工程、控制科学与工程、电气工程等

**推荐使用**：`combined` 字段（已合并去重）

---

## 注意事项

1. **编码问题**：接口返回 UTF-8 编码，浏览器会自动解码显示中文
2. **缓存建议**：这些元数据变化不频繁，前端可以缓存（LocalStorage）减少请求
3. **数据更新**：当数据库中添加新的专业或学校时，接口会自动返回最新数据
4. **专业门类选择**：推荐使用自动补全（autocomplete）而非纯下拉，因为有 573 个选项
