```python
def calculate_recommend_score(
    total_diff,
    single_subject_safety,
    plan_stability_score,
    data_quality_score,
    school_level_score,
    rules
):
    weights = rules["weights"]
    return (
        total_diff * weights["score_diff"] +
        single_subject_safety * weights["single_subject_safety"] +
        plan_stability_score * weights["plan_stability"] +
        data_quality_score * weights["data_quality"] +
        school_level_score * weights["school_level"]
    )
```

#### 4. **分档逻辑**

```python
def classify_rank_type(total_diff, overall_status, row, rules):
    thresholds = rules["rank_thresholds"]
    
    if overall_status == "warning":
        return "rush"  # 压线或接近分数线
    
    if total_diff >= thresholds["safe"]:
        return "safe"  # 明显高出（默认 >= 30 分）
    
    if total_diff >= thresholds["stable"]:
        return "stable"  # 适中（默认 >= 15 分）
    
    return "rush"  # 其他情况
```

**配置**：
```yaml
rank_thresholds:
  safe: 30     # 保底：高出 30 分以上
  stable: 15   # 稳妥：高出 15-30 分
```

**点评**：
- ✅ 分档逻辑简单明确
- ✅ 配置化阈值，易于调整
- ✅ 压线情况自动归入"冲刺"

#### 5. **招生计划稳定性评分**

```python
def calculate_plan_stability_score(plan_history):
    if len(plan_history) < 2:
        return 50.0  # 数据不足
    
    counts = [p["plan_count"] for p in plan_history if p["plan_count"]]
    if len(counts) < 2:
        return 50.0
    
    avg = mean(counts)
    std = pstdev(counts)
    
    if avg == 0:
        return 50.0
    
    cv = std / avg  # 变异系数
    
    # 变异系数越小，稳定性越高
    if cv < 0.1:
        return 90.0
    elif cv < 0.2:
        return 75.0
    elif cv < 0.3:
        return 60.0
    else:
        return 40.0
```

**点评**：
- ✅ 使用变异系数（CV）衡量稳定性
- ✅ 合理的分段评分（CV < 0.1 → 90 分）
- ✅ 数据不足时返回 50 分（中性）

#### 6. **数据质量评分**

```python
def calculate_data_quality_score(row, plan_history):
    score = 100.0
    warnings = []
    
    # 缺少分数线单科信息 -10
    if not row.get("politics_line"):
        score -= 10
        warnings.append("缺少单科分数线")
    
    # 招生计划历史记录少 -20
    if len(plan_history) < 2:
        score -= 20
        warnings.append("近三年招生计划记录不足")
    
    # 缺少当年招生计划 -10
    if not row.get("plan_count"):
        score -= 10
        warnings.append("缺少当年招生计划")
    
    return max(score, 0.0), warnings
```

**点评**：
- ✅ 综合考虑分数线、招生计划的完整性
- ✅ 缺失数据时扣分并给出警告
- ✅ 分数范围：0-100
